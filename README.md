# Time Tools

時間範囲の計算、時間単位の変換、時間の加算・減算ができるシンプルなWebアプリです。
HTML・CSS・JavaScriptのみで構成されているため、GitHub Pagesでそのまま公開できます。

## 機能

### 時間範囲

- 開始時刻と終了時刻から経過時間を計算
- 終了時刻が開始時刻より前の場合は翌日として計算
- 任意の時間を分単位で差し引き
- 結果を「時間・分」「合計分」「小数時間」で表示

### 時間変換

- `hh:mm` から分へ変換
- 分から `hh:mm` へ変換
- どちらの入力欄からでも相互に変換
- 時と分をコロンで区切った入力欄で操作

### 時間の加算・減算

- `hh:mm` 形式の複数の時間をまとめて計算
- 各時間は時と分をコロンで区切った入力欄で操作
- 入力形式のデフォルトを `hh:mm` または `mm`（分）から選択して保存し、未入力の時間にも反映
- 時間ボックスを必要な数だけ追加・削除
- 各ボックス間の演算子を加算または減算から選択
- 結果を `hh:mm` と合計分で表示
- 計算結果がマイナスになる式にも対応

## 使い方

`index.html` をブラウザで開いてください。ビルドや依存パッケージのインストールは不要です。
